HTML Referance Cheat Sheet

TABLE TAGS

HTML_TAG Name Description
<table> Table The wrapper element for all HTML tables.
<thead> Table Head The set of rows defining the colimn headers in a table
<tbody> Table Body The set of rows containing actual table data
<tr> Table Row The table row container
<td> Table Data The table row container
<tfoot> Table Foot The set of rows defining the footer in a table
<link> Link Element Used to link HTML documents to external resources like CSS files


TABLE ATTRIBUTES

Attribute Name Description
colspan Column Span Defines how many columns a td element should span
rowspan Row Span Defines how many rows a td element should span

CSS SELECTORS

SELECTOR NAME DESCRIPTION
p, h, tr Type Selectors Used to match all elements of a given type or tag name.
.calendar-cell Class Selectors The CSS class selector matches elements based on the contents of their class attribute.
#job-title ID Selectors The CSS ID selector matches elements based on the contents of their id attribute.
